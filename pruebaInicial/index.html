Variables y operadores

¿Qué es una variable y para qué sirve?

Es un espacio en memoría que sirve para almacenar información.

¿Cuál es la diferencia entre declarar e inicializar una variable?

Una variable se declara para reservar el espacio necesario en memoria para posteriormente inicializarla, es decir asignarle un valor a ese espacio asignado.

¿Cuál es la diferencia entre sumar números y concatenar strings?

Al sumar números se realiza una operación matemática entre dos variables, mientras que concatenar strings se refiere a unir diferentes cadenas de texto en una sola.

Funciones

¿Qué es una función?

Bloques de código que realizan una actividad específica.

¿Cuándo me sirve usar una función en mi código?

Sirve para modulizar, optimizar, organizar y encapsular nuestro código, o en otras palabras para reutilizar partes del código y hacer que el flujo esté organizado y de esa manera facilitar su mantenimiento o extensión.

¿Cuál es la diferencia entre parámetros y argumentos de una función?

Parámetros son las variables definidas con las cuales se reciben los varoles de una función ejemplo

---Ejemplo:

  • function multiplica(parametro1,parametro2){ return parametro1*parametro2; }

Argumentos son los valores que enviamos a esos parámetros de la función.

---Ejemplo:

  • multiplica(2*3); en donde 2 y 3 serían los argumentos

Condicionales

¿Qué es una condicional?

Son un conjunto de estructuras de control que utilizan comparaciones cuyo resultado es falso o verdadero y, que dependiendo como se haya programado, le permitirá saber a la estructura de control si puede o no acceder a un fragmento de código dentro de su estructura.

¿Qué tipos de condicionales existen en JavaScript y cuáles son sus diferencias?

Las condicionales son: If / Else, Switch, Ternario (Es como un if / else, pero con otra sintaxis más simple)...

¿Puedo combinar funciones y condicionales?

Sí, se pueden combinar funciones y condicionales.

Ciclos

¿Qué es un ciclo?

Es una estructura de control que ejecuta un bloque de instrucciones de manera repetida

¿Qué tipos de ciclos existen en JavaScript?

  • for
  • while
  • do while
  • for in
  • for of

¿Qué es un ciclo infinito y por qué es un problema?

Es un ciclo en el que el condicional para salir del ciclo nunca de cumple, entonces itera de manera indefinida. Podría afectar la memoria del computador, bloquearlo.

Listas

¿Qué es un array?

Una lista de valores que pueden ser de diferente tipo guardadas en una variable, la cual se puede recorrer.

¿Qué es un objeto?

Un objeto es una colección de propiedades, y una propiedad es una asociación entre un nombre (o clave) y un valor. https://developer.mozilla.org/es/docs/Web/JavaScript/Guide/Working_with_Objects

¿Cuándo es mejor usar objetos o arrays?

Objetos es mejor usarlos cuado quieres representar la idea de un "objeto" como tal, como un carro, una persona (que tiene propiedades) es así dentro del algoritmo, el array lo siento más para almacenar una lista de valores.
